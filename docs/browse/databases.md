# 开源大梳理 · databases

## 关系型

- [PostgreSQL](https://github.com/postgres/postgres) — 功能丰富的开源关系型数据库。
- [MySQL](https://github.com/mysql/mysql-server) — 广泛使用的开源关系型数据库。
- [SQLite](https://github.com/sqlite/sqlite) — 嵌入式单文件 SQL 引擎。
- [MariaDB](https://github.com/MariaDB/server) — MySQL 兼容的社区分支。
- [CockroachDB](https://github.com/cockroachdb/cockroach) — 分布式 SQL，兼容 Postgres 协议。
- [TiDB](https://github.com/pingcap/tidb) — MySQL 兼容的分布式 HTAP 数据库。
- [Supabase](https://github.com/supabase/supabase) — 开源 Firebase 替代，底层 Postgres。

## 文档型

- [MongoDB](https://github.com/mongodb/mongo) — 文档型 NoSQL 数据库。

## KV / 缓存

- [Redis](https://github.com/redis/redis) — 内存数据结构存储，常用缓存/队列。
- [KeyDB](https://github.com/Snapchat/KeyDB) — 多线程 Redis 兼容存储。

## 分布式

- [Apache Cassandra](https://github.com/apache/cassandra) — 宽列分布式数据库，高可用写入。
- [ScyllaDB](https://github.com/scylladb/scylladb) — Cassandra 兼容的高性能实现。
- [CockroachDB](https://github.com/cockroachdb/cockroach) — 分布式 SQL，兼容 Postgres 协议。
- [TiDB](https://github.com/pingcap/tidb) — MySQL 兼容的分布式 HTAP 数据库。

## 分析 OLAP

- [ClickHouse](https://github.com/ClickHouse/ClickHouse) — 列式分析型数据库 OLAP。
- [DuckDB](https://github.com/duckdb/duckdb) — 嵌入式分析型数据库。
- [Elasticsearch](https://github.com/elastic/elasticsearch) — 分布式搜索与分析引擎。

## 图数据库

- [Neo4j](https://github.com/neo4j/neo4j) — 原生图数据库。

## 搜索

- [Meilisearch](https://github.com/meilisearch/meilisearch) — 易用的全文搜索引擎。
- [Elasticsearch](https://github.com/elastic/elasticsearch) — 分布式搜索与分析引擎。
- [Typesense](https://github.com/typesense/typesense) — 面向开发者的即时搜索引擎。
