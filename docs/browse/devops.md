# 开源大梳理 · devops 浏览索引

> YAML 投影；权威数据在 data/ graph/

## 容器运行时

- [Docker](https://github.com/docker/cli) — 容器运行时与镜像工作流的事实入口（CLI/引擎生态）。
- [Kubernetes](https://github.com/kubernetes/kubernetes) — 容器编排标准，调度多机工作负载。
- [Traefik](https://github.com/traefik/traefik) — 云原生边缘路由，自动服务发现。
- [Podman](https://github.com/containers/podman) — 无守护进程的容器引擎，Docker 兼容体验。

## 编排

- [Kubernetes](https://github.com/kubernetes/kubernetes) — 容器编排标准，调度多机工作负载。
- [Helm](https://github.com/helm/helm) — Kubernetes 应用包管理器（Chart）。
- [Argo CD](https://github.com/argoproj/argo-cd) — Kubernetes GitOps 持续交付。

## 基础设施即代码

- [Terraform](https://github.com/hashicorp/terraform) — 声明式基础设施即代码，多云 Provider。
- [Pulumi](https://github.com/pulumi/pulumi) — 用通用语言写 IaC（TS/Python 等）。
- [Ansible](https://github.com/ansible/ansible) — 无 Agent 的配置管理与自动化。

## 可观测

- [Prometheus](https://github.com/prometheus/prometheus) — 指标采集与告警的云原生监控核心。
- [Grafana](https://github.com/grafana/grafana) — 可观测可视化与仪表盘。

## 网络与网关

- [nginx](https://github.com/nginx/nginx) — 高性能反向代理与 Web 服务器。
- [Traefik](https://github.com/traefik/traefik) — 云原生边缘路由，自动服务发现。

## CI / GitOps

- [actions/runner](https://github.com/actions/runner) — GitHub Actions 自托管 Runner。
- [act](https://github.com/nektos/act) — 本地运行 GitHub Actions 工作流。
- [Argo CD](https://github.com/argoproj/argo-cd) — Kubernetes GitOps 持续交付。
