# 开源大梳理 · devtools

## 搜索与发现

- [ripgrep](https://github.com/BurntSushi/ripgrep) — 极速递归正则搜索，默认尊重 .gitignore。
- [fd](https://github.com/sharkdp/fd) — 用户友好的 find 替代，默认忽略隐藏与 gitignore。
- [fzf](https://github.com/junegunn/fzf) — 通用模糊查找器，可嵌入 shell 工作流。

## 阅读与 Diff

- [bat](https://github.com/sharkdp/bat) — 带语法高亮与 Git 集成的 cat 替代。
- [delta](https://github.com/dandavison/delta) — 语法高亮的 git/diff 阅读器。

## 数据 CLI

- [jq](https://github.com/jqlang/jq) — 命令行 JSON 处理器。
- [yq](https://github.com/mikefarah/yq) — 类 jq 的 YAML/JSON/XML 处理器。

## 任务与版本

- [just](https://github.com/casey/just) — 现代命令运行器，用 justfile 替代散落 makefile 目标。
- [mise](https://github.com/jdx/mise) — 多语言运行时与工具版本管理（asdf 类）。

## Git/Docker TUI

- [lazygit](https://github.com/jesseduffield/lazygit) — 终端 Git TUI，降低复杂 git 操作成本。
- [lazydocker](https://github.com/jesseduffield/lazydocker) — Docker/Compose 的终端仪表盘。

## 基准

- [hyperfine](https://github.com/sharkdp/hyperfine) — 命令行基准测试工具。

## 语言工具链

- [ruff](https://github.com/astral-sh/ruff) — 极速 Python linter/formatter。
- [uv](https://github.com/astral-sh/uv) — 极速 Python 包与项目管理器。
- [biome](https://github.com/biomejs/biome) — 快速 JS/TS linter+formatter（Rust）。

## 编辑器与解析

- [helix](https://github.com/helix-editor/helix) — 模态编辑器，内建 LSP/tree-sitter。
- [Neovim](https://github.com/neovim/neovim) — 可扩展的 Vim 分支，Lua 配置与 LSP 成熟。
- [tree-sitter](https://github.com/tree-sitter/tree-sitter) — 增量解析库，驱动编辑器语法与结构化编辑。
