# 开源大梳理 · backend

## Python

- [FastAPI](https://github.com/fastapi/fastapi) — 高性能 Python API 框架，类型提示与 OpenAPI 一流。
- [Django](https://github.com/django/django) — 电池齐全的 Python Web 框架。
- [Flask](https://github.com/pallets/flask) — 轻量 Python Web 微框架。

## Node / TS

- [Express](https://github.com/expressjs/express) — Node.js 最流行的 HTTP 框架。
- [NestJS](https://github.com/nestjs/nest) — Angular 风格的 Node 企业框架。
- [Hono](https://github.com/honojs/hono) — 超轻量多运行时 Web 框架（Edge/Node/Bun）。
- [Prisma](https://github.com/prisma/prisma) — TypeScript ORM / 数据访问层。

## Go

- [Gin](https://github.com/gin-gonic/gin) — 高性能 Go HTTP 框架。
- [Fiber](https://github.com/gofiber/fiber) — Express 风格的 Go Web 框架。

## Rust

- [Axum](https://github.com/tokio-rs/axum) — 基于 Tokio 的 Rust Web 框架。
- [Actix Web](https://github.com/actix/actix-web) — 高性能 Rust HTTP 框架。

## JVM

- [Spring Boot](https://github.com/spring-projects/spring-boot) — Java 企业应用快速启动框架。

## Ruby

- [Ruby on Rails](https://github.com/rails/rails) — 约定优于配置的 Ruby 全栈框架。

## PHP

- [Laravel](https://github.com/laravel/laravel) — 优雅的 PHP Web 框架。

## RPC

- [gRPC-Go](https://github.com/grpc/grpc-go) — Go 的 gRPC 实现。
- [Connect](https://github.com/connectrpc/connect-go) — 简易 Protobuf RPC，兼容 gRPC。

## 队列与流

- [Celery](https://github.com/celery/celery) — Python 分布式任务队列。
- [BullMQ](https://github.com/taskforcesh/bullmq) — 基于 Redis 的 Node 任务队列。
- [Apache Kafka](https://github.com/apache/kafka) — 分布式流处理与日志总线。
- [RabbitMQ](https://github.com/rabbitmq/rabbitmq-server) — 成熟的消息代理。
