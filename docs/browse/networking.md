# 开源大梳理 · networking

## 代理与负载均衡

- [Envoy](https://github.com/envoyproxy/envoy) — 云原生高性能代理，常作服务网格数据面。
- [HAProxy](https://github.com/haproxy/haproxy) — 高可用负载均衡与代理。

## API 网关

- [Apache APISIX](https://github.com/apache/apisix) — 动态云原生 API 网关。
- [Kong](https://github.com/Kong/kong) — 基于 OpenResty 的 API 网关。

## 网格与 CNI

- [Linkerd](https://github.com/linkerd/linkerd2) — 超轻量 Kubernetes 服务网格。
- [Cilium](https://github.com/cilium/cilium) — 基于 eBPF 的云原生网络与安全。

## DNS

- [CoreDNS](https://github.com/coredns/coredns) — 可插件化的 DNS 服务器，K8s 默认 DNS。
- [Unbound](https://github.com/NLnetLabs/unbound) — 验证型递归 DNS 解析器。

## 组网 VPN

- [Tailscale](https://github.com/tailscale/tailscale) — 基于 WireGuard 的零配置组网。
- [Headscale](https://github.com/juanfont/headscale) — Tailscale 控制服务器的开源自托管实现。
- [NetBird](https://github.com/netbirdio/netbird) — 基于 WireGuard 的开源零信任组网。

## 隧道与穿透

- [frp](https://github.com/fatedier/frp) — 内网穿透，暴露本地服务到公网。
- [cloudflared](https://github.com/cloudflare/cloudflared) — Cloudflare Tunnel 客户端。
- [GOST](https://github.com/go-gost/gost) — GO 语言的安全隧道与代理工具集。

## 排障与测量

- [mitmproxy](https://github.com/mitmproxy/mitmproxy) — 交互式 HTTPS 中间人代理与流量检查。
- [Wireshark](https://github.com/wireshark/wireshark) — 网络协议分析器。
- [Nmap](https://github.com/nmap/nmap) — 网络发现与安全审计扫描器。
- [iperf3](https://github.com/esnet/iperf) — 网络带宽性能测试工具。


> 相关：`nginx`/`caddy`/`traefik`/`istio` 见 devops；`wireguard` 见 security。
