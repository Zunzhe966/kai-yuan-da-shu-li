# 开源大梳理 · desktop

## WebView / 浏览器壳

- [Electron](https://github.com/electron/electron) — 用 Chromium+Node 构建跨平台桌面应用。
- [Tauri](https://github.com/tauri-apps/tauri) — Rust 后端 + 系统 WebView 的轻量桌面框架。
- [Wails](https://github.com/wailsapp/wails) — Go + 系统 WebView 构建桌面应用。
- [Neutralinojs](https://github.com/neutralinojs/neutralinojs) — 轻量跨平台应用框架，避免捆绑完整浏览器。

## 原生 UI

- [Qt](https://github.com/qt/qtbase) — 成熟的跨平台原生 UI 框架。
- [GTK](https://github.com/GNOME/gtk) — GNOME 系跨平台 GUI 工具包。
- [wxWidgets](https://github.com/wxWidgets/wxWidgets) — 原生控件外观的跨平台 C++ GUI 库。
- [Slint](https://github.com/slint-ui/slint) — 声明式原生 UI 工具包，多语言绑定。
- [Iced](https://github.com/iced-rs/iced) — Rust 的 Elm 风格跨平台 GUI 库。

## 即时模式

- [Dear ImGui](https://github.com/ocornut/imgui) — 即时模式 GUI，适合工具/调试/游戏内嵌面板。
- [egui](https://github.com/emilk/egui) — Rust 即时模式 GUI，易嵌入工具与游戏。
- [Dear PyGui](https://github.com/hoffstadt/DearPyGui) — Python 即时模式 GPU 加速 GUI。

## 多端框架桌面目标

- [Flutter Desktop](https://github.com/flutter/flutter) — Flutter 的桌面目标（Windows/macOS/Linux）。
- [Compose Multiplatform](https://github.com/JetBrains/compose-multiplatform) — Jetpack Compose 多平台，含桌面目标。
- [Avalonia](https://github.com/AvaloniaUI/Avalonia) — 跨平台 .NET UI 框架。
- [Uno Platform](https://github.com/unoplatform/uno) — 用 C#/XAML 做跨平台应用（含桌面）。

## Python GUI

- [PyQt](https://github.com/Python-PyQt/PyQt) — Qt 的 Python 绑定（商业许可需注意）。
- [PySide (Qt for Python)](https://github.com/qtproject/pyside-setup) — Qt 官方 Python 绑定。
- [Dear PyGui](https://github.com/hoffstadt/DearPyGui) — Python 即时模式 GPU 加速 GUI。
