# 开源大梳理 · web-frontend

## UI 框架 / 状态

- [React](https://github.com/facebook/react) — 组件化 UI 库，Web 前端事实标准之一。
- [Vue](https://github.com/vuejs/core) — 渐进式前端框架，模板友好。
- [Svelte](https://github.com/sveltejs/svelte) — 编译型 UI 框架，运行时更薄。
- [Radix Primitives](https://github.com/radix-ui/primitives) — 无样式无障碍头组件原语（React）。
- [Zustand](https://github.com/pmndrs/zustand) — 轻量 React 状态库。
- [Redux Toolkit](https://github.com/reduxjs/redux-toolkit) — 官方推荐的 Redux 写法与工具集。
- [TanStack Query](https://github.com/TanStack/query) — 服务端状态/异步数据缓存库。

## 元框架

- [Next.js](https://github.com/vercel/next.js) — React 全栈框架，SSR/App Router。
- [Nuxt](https://github.com/nuxt/nuxt) — Vue 元框架，SSR/文件系统路由。
- [Remix](https://github.com/remix-run/remix) — 以 Web 标准为中心的 React 框架。
- [Astro](https://github.com/withastro/astro) — 内容站点优先的多框架元框架。

## 工程化

- [Vite](https://github.com/vitejs/vite) — 新一代前端构建与开发服务器。
- [webpack](https://github.com/webpack/webpack) — 成熟模块打包器，插件生态大。
- [Storybook](https://github.com/storybookjs/storybook) — 组件开发与文档工坊。
- [Turborepo](https://github.com/vercel/turborepo) — JS/TS monorepo 构建系统。
- [pnpm](https://github.com/pnpm/pnpm) — 节省磁盘的包管理器，对 monorepo 友好。

## 样式与组件原语

- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss) — 工具类优先的 CSS 框架。
- [Radix Primitives](https://github.com/radix-ui/primitives) — 无样式无障碍头组件原语（React）。

## 测试与文档

- [Playwright](https://github.com/microsoft/playwright) — 现代多浏览器 E2E 测试。
- [Cypress](https://github.com/cypress-io/cypress) — 开发者友好的 E2E 测试工具。
- [Vitest](https://github.com/vitest-dev/vitest) — 与 Vite 同构的单元测试框架。
- [Storybook](https://github.com/storybookjs/storybook) — 组件开发与文档工坊。
